# Python Basics Cheat Sheet

Welcome ! This repo is a study guide for new programming/coding students.
It is designed to be beginner-friendly and organized by topics.

---

### How to use

Each folder covers Python topics (variables, loops, functions, etc.)

- Open any ".py" file to view
- A short defintion written for each example
- Usage example
- Occasional tips

---

### Folder Overview

| Folder | Topic |
| '01-basics' | variables, operators, data types |
| '02-control_flow' | conditionals, loops |
| '03-functions' | defining/calling functions |
| '04-collections' | lists, tuples, sets, dictionaries |
| '05-strings' | string methods and formatting |
| '06-files_exceptions' | file/error handling |
| '07-modules_packages' | importing/using libraries |
| '08-extra' | tricks and common patterns |

---

### Tips

- Read one file at a time
- Run examples in VSCode or a Python app (like Spyder)
- Try changing/personalizing the code to see what happens
